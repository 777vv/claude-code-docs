## 真实使用示意

### 学生场景

```
07:00  📚 AI 早起播报
       "今天 3 节课: 高数 / 英语 / 操作系统
        昨天高数作业你没做完, 第 3 题卡在哪了?
        英语单词复习: 5 个老词 (上次错的)"

10:00  📝 课间问 AI:
       "刚老师讲的拉格朗日中值定理我没听懂"
       → AI 用大白话 + 类比讲解

19:00  🎓 晚自习
       "考你今天学的"
       → AI 出 5 题 → 你答 → AI 判分 + 解释

22:00  💤 睡前
       AI 自动汇报今日:
       - 学了什么
       - 错题 3 个 (已加进错题本)
       - 明天重点复习这些
```

### 家庭场景（妈妈用）

```
微信家庭群:
妈妈: "晚上买番茄和鸡蛋"
AI: "✅ 加入购物清单 (今日)"

爸爸: "周三要交水电费啊"
AI: "✅ 已加提醒 周三 9:00"

孩子: "妈妈我数学题不会"
AI: "@妈妈 孩子问数学题, 你方便时回他"
    "(同时给孩子讲了基础思路)"

周日晚:
AI 自动汇报:
"本周家务统计:
- 爸爸: 倒垃圾×7, 接孩子×3
- 妈妈: 做饭×14
- 建议下周爸爸做 2 顿减负"
```

### 老人场景（子女配置好后给老人用）

```
老人对着微信语音说:
"我今天血压有点高"
→ AI 记录健康数据
→ 同步给子女
→ 提醒老人下次量血压时间

"什么时候交养老金"
→ AI 查日历回答
→ 提前 3 天再提醒

"想老二了"
→ AI 帮老人发消息给二儿子
→ 不需要老人会用复杂功能
```

## 常见担忧

**Q：我电脑不好，能跑动吗？**
A：能。AI "大脑"在云端跑（DeepSeek 服务器），你电脑只是"接口"，**老笔记本完全够**。如果想纯本地（Ollama）那需要好显卡。

**Q：我真的看不懂技术教程**
A：本站教程**给小白看的**——每一步都告诉你"输入什么"、"会看到什么"。卡住了进群问，作者真人在线。

**Q：会不会泄露我家事 / 学习记录？**
A：用 **DeepSeek API** 是国内合规的，你的数据归你；用 **Ollama 本地模型** 数据完全不出你电脑（最安全）。本站 [安全清单](/openclaw/ops/security-checklist) 详讲。

**Q：会被 AI 替代吗？**
A：你做的事 AI 不会替代——AI 替代的是"找资料 / 排日程 / 写流水账"。**你陪家人的时间、独立思考、做选择——AI 不行**。

**Q：孩子可以用吗？**
A：可以，但 [home/家庭管家](/openclaw/cases/family-manager) 里有给孩子的安全规则（不能让 AI 教孩子查危险内容、不能让孩子自己下单买东西等）。

**Q：老人能用吗？**
A：能。配好后老人对着微信语音说话就行——不用学 APP，**自然语言交互**就是给老人设计的最友好形式。
